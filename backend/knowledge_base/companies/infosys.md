# Infosys — Placement Guide for LPU Students

## Overview
Infosys is a top IT recruiter at LPU, hiring through multiple tracks via the **InfyTQ** platform.

## Roles & CTC

| Role | CTC |
|---|---|
| Systems Engineer (SE) | 3.6 LPA |
| Digital Specialist Engineer (DSE) | 8–9 LPA |
| Power Programmer | 11 LPA |

## Eligibility
- 60%+ in 10th, 12th, and graduation (no rounding)
- No current active backlogs
- All CS/IT/ECE branches preferred; other branches considered for SE

## Selection Process

### Step 1 — InfyTQ / HackWithInfy Assessment
- **Platform**: infytq.infosys.com
- **Sections**: Pseudocode, Core CS, Programming Fundamentals, Database
- DSE/Power track: **3 coding problems in 3 hours** (medium difficulty)
- SE track: Basic aptitude + 1 easy coding problem

### Step 2 — Technical Interview (40 min)
- Projects deep-dive (know every line of your project)
- OOPS concepts: Classes, Inheritance, Polymorphism with code
- Java or Python basics (Infosys prefers Java)
- SQL: Write JOINs, GROUP BY queries on the spot
- OS basics: Process vs Thread, deadlock
- DSA: Arrays, linked lists, sorting

### Step 3 — Managerial Round (20 min)
- Team coordination, leadership, relocation willingness
- Situational questions: "How would you handle a deadline miss?"

### Step 4 — HR Round (15 min)
- Bond: 2-year service agreement
- CTC discussion, start date, location preference

## DSE Coding Focus
- Graph BFS/DFS problems
- Medium-level DP (0-1 knapsack, LCS)
- String manipulation
- Tree traversals

## Prep Tips
1. Complete InfyTQ free certification — it boosts your profile
2. Practice Java OOP heavily (Infosys uses Java internally)
3. Know SQL JOINs and aggregates thoroughly
4. Solve 50+ medium LeetCode before DSE attempt
5. Prepare 2–3 project descriptions clearly
6. Practice pseudocode reading — specific to InfyTQ format
7. Timeline: Infosys typically visits LPU January–March
